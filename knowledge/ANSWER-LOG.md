# Headache Answer Log

A running record of the questions `/headache-helper` has answered and the answers it gave, with the exact wiki sources each answer relied on. `/headache-helper` appends here after every answer; you can also add entries by hand.

**Why this exists:** when `/headache-ingest` approves new material into the wiki, it re-checks every question here against the updated wiki. If a newer or higher-tier source changes a previously-given answer, that entry is flagged — old answer, new answer, and the source that caused the change — so you can see exactly which past guidance no longer holds.
